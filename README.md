# Genetics Inheritance Explorer

## Overview
A simple Java Swing application to visualize offspring genotype probabilities from parent genotypes.

## Features
- Input parent genotypes
- Calculate possible offspring genotypes
- Display probabilities

## Technologies
Java SE, Swing

## Setup
1. Download or clone the repo.
2. Compile with `javac *.java`.
3. Run `java com.yourname.GeneticsExplorer`.

## Usage
Enter genotype strings (e.g., AaBb) and click Calculate.
